◆このゲームについて
迷路×アスレチックという2つのジャンルを融合した3Dアクションシューティングゲーム
様々なイベントやギミックを潜り抜け、出口(はしご)を目指せ！

◆起動方法
Frozen Frontier Ver.Masterフォルダー内の、ProjectC++_phase2.exeをダブルクリックすると起動します。

◆動作確認済み環境
Windows 11

◆基本操作

//=====キーボード============
◆全体
Escキー ゲーム終了

◆インゲーム中
Wキー 上移動
Aキー 左移動
Sキー 下移動
Dキー 右移動

スペースキー ジャンプ

Jキー 火炎弾発射 (Mキーから変更)
Jキー長押し 充填式火炎弾発射 (Mキーから変更)
//===========================


//=====ゲームパッド(XInput方式)============
◆インゲーム中
Lスティック 上下左右移動

十字キー上 上移動
十字キー左 左移動
十字キー下 下移動
十字キー右 右移動

Aボタン ジャンプ

RBボタン 火炎弾発射
RBボタン長押し 充填式火炎弾発射

STARTボタン ポーズ画面表示

※チュートリアルステージ
Rスティック ゲーム説明UI切り替え

◆ポーズ画面
Lスティック 左右移動
十字キー上 上移動
十字キー下 下移動
Aボタン 決定

◆ステージセレクト画面
Lスティック 左右移動
十字キー左 左移動
十字キー右 右移動
Aボタン 決定

◆それ以外の場面
Aボタン 各画面遷移
F3キー ボーダーレスフルスクリーン表示切替
//=========================================

◆実装内容
[プレイヤー関連処理]
xファイル(3Dモデル)をtxtファイルでファイルパスを参照して表示
xファイルの複数パーツ表示(モデルの親子関係)
移動処理
回転処理
ジャンプ処理 (ベータ版で実装)

[球関連処理]
通常弾（火炎弾）の処理
長押しで敵オブジェクトを貫通する弾の処理(充填式火炎弾)

[ブロック関連処理]
プレイヤーとの当たり判定 (触れたらプレイヤーの座標を補正)
プレイヤーの弾をあてるとブロックが非表示(破壊)されるブロックの実装
移動しているブロックの上に乗るとそのまま上に乗れるブロックの実装


[UI関連処理]
充填式火炎弾が充填完了になった際にそれを知らせるUIの実装
体力回復した際にそれを知らせるUIの実装
各種アイテムを取得し、プレイヤーにバフ状態が付与されていた場合、付与状況を知らせるUIの実装
ゲーム開始前のスタートコールUI表示を実装。


[その他]
タイマー(インゲーム内)の実装
タイトル画面の実装
スコアの実装
リザルト画面の実装＆スコアをリザルト画面で表示する処理の実装 
フェードイン、フェードアウトをして画面遷移を実行する処理の実装
txtファイルでのオブジェクト配置情報、タイマーの取得
ゴールマーカーの実装
ステージセレクト画面の実装

ポーズ画面の実装 (PreMaster版で実装)
ボーダーレスフルスクリーン表示の実装 (PreMaster版で実装)
カメラが振動する処理の実装
サウンド関係の処理の実装
チュートリアル専用のUI表示(ゲーム説明を実装)
ゲームパッドのスティック動作（プレス、トリガー）を実装。

◆注意事項
実行前に必ずdataファイルがディレクトリ内に存在するか確認してください！
プレイヤーの体力が0になるか、タイマーの残り秒数が0になるか、ゴールマーカーに触れると自動的にリザルト画面遷移します。
